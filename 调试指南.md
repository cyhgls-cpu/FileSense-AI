## 🐛 常见问题修复

### 问题 1: 时间始终显示"未知"

**可能原因 1**: 文件系统不支持
- 某些网络驱动器不提供 birthtime
- FAT32/exFAT 可能只有 mtime

**测试方法**:
```javascript
// 在控制台执行
const testDate = new Date("2026-03-14T15:30:45.123Z");
console.log('测试日期解析:', testDate);
console.log('是否有效:', !isNaN(testDate.getTime()));
console.log('格式化:', formatDate("2026-03-14T15:30:45.123Z"));
```

**可能原因 2**: 代码未重新加载
**解决**: 完全重启应用（Ctrl+Shift+R 强制刷新）

### 问题 2: 预览区域空白

**可能原因 1**: 不是图片/视频格式
**检查**: 文件扩展名是否在支持列表中

**可能原因 2**: 缩略图生成失败
**检查**: 控制台是否有 `[预览] 原图缩略图结果：失败`

**可能原因 3**: img 元素未创建
**检查**: HTML 模板中是否正确创建了 img 标签

### 问题 3: 控制台完全没有日志

**可能原因**: 应用未正确启动或代码未更新
**解决**:
```bash
# 完全重启
taskkill /F /IM electron.exe
npm start
```

## 🔬 高级调试技巧

### 技巧 1: 实时查看数据流

在控制台中执行：
```javascript
// 监听 IPC 调用
const originalInvoke = ipcRenderer.invoke;
ipcRenderer.invoke = async function(channel, ...args) {
  console.log('[IPC 调用]', channel, args);
  return originalInvoke.call(this, channel, ...args)
    .then(result => {
      console.log('[IPC 返回]', channel, result);
      return result;
    })
    .catch(err => {
      console.error('[IPC 错误]', channel, err);
      throw err;
    });
};
```

### 技巧 2: 手动测试 formatDate

```javascript
// 在控制台执行
```
