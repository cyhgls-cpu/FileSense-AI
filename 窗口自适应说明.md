# 🪟 窗口自适应大小功能说明

## ✨ 新增功能

软件窗口现在支持**自由调整大小**，文件多了也不怕！

---

## 🎯 调整方法

### 方法 1: 拖动窗口边缘（推荐）

将鼠标移到窗口边缘或角落：
- ↔️ **左右边缘**: 调整宽度
- ↕️ **上下边缘**: 调整高度
- ⤡ **四个角落**: 同时调整宽度和高度

当鼠标变成双向箭头时，按住左键拖动即可。

### 方法 2: 最大化窗口

点击窗口右上角的 **□** 按钮，窗口将全屏显示。

### 方法 3: 双击标题栏

双击窗口顶部的标题栏，可以快速最大化/还原窗口。

---

## 📐 尺寸限制

### 最小尺寸
- **宽度**: 1000px - 保证内容正常显示
- **高度**: 700px - 保证按钮可见

### 初始尺寸
- **宽度**: 1400px - 更宽的视野
- **高度**: 900px - 更多的垂直空间

### 最大尺寸
- **无限制** - 可以根据您的显示器自由调整

## 🎨 自适应优化

### 1. 主内容区域

```css
grid-template-columns: repeat(auto-fit, minmax(500px, 1fr));
```

**效果**:
- 窗口宽度 > 1000px: 显示 2 列
- 窗口宽度 > 1500px: 可能显示 3 列
- 窗口宽度 < 1000px: 自动变为 1 列

### 2. 文件列表高度

```css
max-height: calc(100vh - 400px);
```

**效果**:
- 窗口越高，列表区域越高
- 窗口越矮，列表区域越矮
- 始终保持顶部和底部控件可见

### 3. 美化滚动条

当文件很多时，滚动条会自动出现：

**样式**:
- 🎨 马卡龙配色（粉色渐变）
- 📏 细滚动条（8px）
- ✨ 悬停高亮效果

## 🔧 技术细节

### Electron 窗口配置

```javascript
mainWindow = new BrowserWindow({
  width: 1400,           // 初始宽度
  height: 900,           // 初始高度
  minWidth: 1000,        // 最小宽度
  minHeight: 700,        // 最小高度
  resizable: true,       // 允许调整大小
  maximizable: true,     // 允许最大化
  minimizable: true,     // 允许最小化
});
```

### CSS 自适应布局

#### Grid 自适应列数
```css
.main-content {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(500px, 1fr));
}
```

**解释**:
- `auto-fit`: 自动适应列数
- `minmax(500px, 1fr)`: 每列最小 500px，最大占满剩余空间

#### 高度自适应
```css
.file-list {
  max-height: calc(100vh - 400px);
}
```

**解释**:
- `100vh`: 窗口高度
- `- 400px`: 减去头部、底部等固定高度
- 剩余空间都给列表

#### 响应式容器
```css
.container {
  width: calc(100% - 40px);
  max-width: 100%;
}
```

**解释**:
- 留出左右各 20px 边距
- 不限制最大宽度，可以占满窗口

## 📊 性能考虑

### 大量文件（> 1000 个）

**建议**:

## ⚠️ 注意事项

### 1. 不要太小

虽然可以调整到 1000x700，但再小就会影响使用体验：
- 按钮可能重叠
- 文字可能被截断
- 布局可能错乱

### 2. 最佳尺寸

根据测试，以下尺寸最舒适：

**1080p 显示器**:
- 推荐：1400x900 或最大化

**2K 显示器**:
- 推荐：1800x1100 或最大化

**4K 显示器**:
- 推荐：最大化或 2560x1440

### 3. 多显示器

如果您有多个显示器：
- 可以拖动到任意显示器
- 每个显示器独立记忆大小
- 支持跨显示器移动
