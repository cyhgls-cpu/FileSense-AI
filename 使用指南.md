## 🔍 故障排除

### 如果应用无法启动

#### 症状 1：完全没反应
**解决**：
1. 检查任务管理器是否有 `electron.exe` 进程
2. 如果有，结束进程后重新启动
3. 双击 `一键启动.cmd` 再次尝试

#### 症状 2：窗口闪退
**解决**：
```cmd
cd "k:\AI soft\rerere"
set PATH=C:\Program Files\nodejs;%PATH%
npx electron . --dev
```
按 F12 查看开发者工具中的错误

#### 症状 3：报错找不到模块
**解决**：
```cmd
cd "k:\AI soft\rerere"
"C:\Program Files\nodejs\npm.cmd" install
```

## 💡 使用技巧

### 快速选择要删除的文件
1. 每组重复文件都有两个选项
2. 默认选中"保留此文件"
3. 如果要删除另一个，点击对应的单选按钮
4. 勾选"标记为待删除"
5. 最后点击"删除选中文件"

### 批量操作
1. 使用全选复选框选择所有组
2. 一次性删除所有选中的重复文件
3. 删除前会弹出确认对话框

### 安全删除
- 每次删除都会显示确认对话框
- 显示将要删除的文件列表
- 显示总大小
- 确认后才会执行删除

