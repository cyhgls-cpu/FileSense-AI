## 🔍 问题排查

### 如果图片缩略图不显示
1. 检查是否安装了 sharp：
   ```bash
   cd "k:\AI soft\rerere"
   npm install sharp
   ```
2. 重启应用

### 如果视频无法播放
- 确保系统有相应的视频解码器
- 可以尝试点击"打开文件位置"手动打开

### 如果时间仍然显示"未知"
- 这可能是文件系统不支持创建时间（某些网络驱动器）
- 本地 NTFS 文件系统应该没问题
