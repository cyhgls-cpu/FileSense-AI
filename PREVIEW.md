## 🎯 交互流程示例

### 场景 1: 用户首次启动

```
1. 应用启动
   ↓
2. 自动硬件探测
   ↓
3. 检测到 RTX 5070 Ti + 32GB RAM
   ↓
4. 推荐"平衡模式"或"完全 AI 模式"
   ↓
5. 用户点击"保存"
   ↓
6. 后台开始下载推荐模型（显示进度条）
   ↓
7. 下载完成，立即可用
```

### 场景 2: 手动切换模式

```
用户操作：点击"完全 AI 模式"卡片
    ↓
UI 反馈：卡片高亮，显示"已选择"标记
    ↓
弹窗提示：
"切换到完全 AI 模式需要额外下载 1.1GB 模型文件
 预计耗时 5-10 分钟（取决于网络速度）"
    ↓
用户确认 → 开始下载
用户取消 → 保持当前模式
```

### 场景 3: 模型懒加载演示

```
时间线:
00:00 - 用户点击"AI 分析"按钮
00:01 - 加载 LLM 模型（进度条：正在加载...）
00:03 - 模型加载完成，开始分析
00:05 - 分析完成，显示结果
00:05~01:05 - 模型保持在内存中（待命状态）
01:05 - 60 秒超时触发
01:06 - 自动卸载模型，释放 1.5GB 内存
01:06 - UI 提示："已释放模型内存"
```

## 🎨 马卡龙色系设计

### 配色方案

| 元素 | 颜色代码 | 视觉效果 |
|------|---------|---------|
| 背景渐变 | `#FFB7C5` → `#E6E6FA` → `#AAF0D1` | 粉→紫→薄荷 |
| 卡片背景 | `rgba(255,255,255,0.9)` | 半透明白色 |
| 选中边框 | `#FFB7C5` (粉色) | 温柔高亮 |
| 按钮渐变 | `#FFB7C5` → `#FFDAB9` | 粉→桃色 |
| 统计卡片 | `#FFFACD` → `#B0E0E6` | 柠檬→天蓝 |

### 动画效果

```css
/* 卡片悬停 */
.mode-card:hover {
  transform: translateY(-5px);
  box-shadow: 0 8px 20px rgba(0,0,0,0.1);
}

/* 选中状态 */
.mode-card.selected {
  border-color: #FFB7C5;
  background: linear-gradient(135deg, #fff, #ffb7c5);
}

/* 通知滑入 */
@keyframes slideIn {
  from { transform: translateX(400px); opacity: 0; }
  to { transform: translateX(0); opacity: 1; }
```
